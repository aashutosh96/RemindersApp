# RemindersApp

### Features
- Create Reminder 
- Select Datetime
- Edit Reminder title and due date
- Toggle Reminder Complete with Checkbox
- Slide to delete reminders
- Sort Reminders in Ascending/Descending Order
- Filter Complete and Due Reminders
